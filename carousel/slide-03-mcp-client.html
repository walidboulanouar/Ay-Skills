mcp-
client

Connect Claude to any MCP server without killing context.

  • Loads tool schemas on demand. Not 2000 tokens upfront at session start.
  • Works with Zapier, GitHub, n8n, filesystem, or any MCP server you configure.
  • Three commands: servers / tools / call. Progressive disclosure by design.
connecting to MCP servers
$ python mcp_client.py servers
zapier connected
github connected
n8n connected
filesystem connected

$ python mcp_client.py tools zapier
847 tools available
gmail.send
sheets.append
slack.message
...

$ python mcp_client.py call
zapier gmail.send
'{"to":"...","body":"..."}'
sent.
